When creating a QueryParamGroup using QueryParamBuilder#group you can pass further configuration using its second parameter.

Router Options

Define how ngqp interacts with the Angular Router. These options are simply forwarded to the router. To learn more about how they work consult the Angular documentation. If not specified, ngqp uses the global configuration.

replaceUrl

Defines whether updating the URL due to a change in the model adds or replaces the state in the URL. This essentially defines whether the user can undo it via the back button.

preserveFragment

Defines whether a possibly set fragment in the URL should be preserved.
