When importing QueryParamModule you can use the withConfig method to change the global configuration of the module.

Router Options

Use the routerOptions configuration key to define how ngqp interacts with the Angular Router. These options are simply passed along to the router. To learn more about how they work consult the Angular documentation.

replaceUrl

Defines whether updating the URL due to a change in the model adds or replaces the state in the URL. This essentially defines whether the user can undo it via the back button. By default, ngqp sets this to true, i.e., changing the value of the model does not add a new state to the history.

Each QueryParamGroup instance can additionally overwrite this configuration.

preserveFragment

Defines whether a possibly set fragment in the URL should be preserved. By default, ngqp sets this to true.

Each QueryParamGroup instance can additionally overwrite this configuration.
