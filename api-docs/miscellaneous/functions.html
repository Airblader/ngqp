createEmptyOnDeserializer
createEmptyOnDeserializer(deserializer, emptyOn)

Wraps a deserializer which defaults to the given value.

This function returns a new deserializer which mirrors the given deserializer, except that on a null value to deserialize, the given default value is returned instead.

Parameters :
Name Optional Description
deserializer No

The deserializer to wrap and mirror.
emptyOn No

The default value to return if the value to deserialize is null.
Returns : ParamDeserializer<T>
createEmptyOnSerializer
createEmptyOnSerializer(serializer, emptyOn, compareWith)

Wraps a serializer which defaults to the given value.

This function returns a new serializer which mirrors the given serializer, except that if the given default value equals the value to be serialized (using the provided comparator), null is returned instead.

Parameters :
Name Optional Description
serializer No

The serializer to wrap and mirror.
emptyOn No

The default value for which the serialization should return null.
compareWith No

The comparator function to compare two values.
Returns : ParamSerializer<T>
